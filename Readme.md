## Getting Started

### Creating a Project
Clone the CCDG Scorekeeping 2025 reposistory from github into your local machine.

### Virual Environment
Create a venv running on Python 3.12 or greater.  Use pip to install the following dependencies in your new venv:
 * sqlalchemy
 * pandas
 * gspread
 * pydrive2
 * openpyxl
